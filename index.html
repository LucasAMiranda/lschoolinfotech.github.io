Curso de Programação para Iniciante

Aprenda a programar neste curso, ele foi pensado para quem nunca mexeu com programação e tem vontade de conhecer este mundo. Por isso, neste curso não vamos focar em nenhuma linguagem de programação de computador, mas conceitos e fundamentos gerais que todo o programador necessita saber.

Além disso, você aprenderá como modelar dados usando UML para a montagem do banco de dados com comandos de SQL para consulta, criação de telas e muito mais, o que facilitará o progresso em uma linguagem de programação específica posteriormente.
